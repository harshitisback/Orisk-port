Our Idea

The application "0RISK" will help companies/businesses in ensuring the health and safety of their workers. The app stores data related to the medical history of the user and predicts if a user is fit to perform the job. It also has an SOS message-sending button so emergency services reach on time. The application is for the android operating system and is developed using Java programming language so It supports numerous safety features. It can be used in different business practices like on-site work practices (e.g. construction, electrical work) or even work involving sitting in office and working on computer.
